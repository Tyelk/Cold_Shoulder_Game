# Cold Shoulder

Cold Shoulder is a simple top-down driving game built using Python and Pygame-ce.
It was created to help demonstrate the difficulty of following the road rules while sharing the road with cyclists.

## How To Setup
***Note:*** Make sure Python is installed! (Python 3.13+ recommended)

### 1. Clone the Repository
```
git clone https://github.com/Tyelk/Cold_Shoulder_Game.git
cd Cold_Shoulder_Game
```

### 2. Install Pygame-ce

```
pip install pygame-ce
```

### 3. Run the Game

```
python code/main.py
```

## How To Play

The objective of the game is to reach the delectable sweet treat while following the road rules as closely as possible. You are graded on how well you follow the road rules.

| Action | Key |
| :---: | :---: |
| Accelerate  | `W`  |
| Brake/Reverse | `S` |
| Steer Left | `A` |
| Steer Right | `D` |
| Toggle Left Indicator | `Q` |
| Toggle Right Indicator | `E` |
