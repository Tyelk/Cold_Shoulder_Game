# Cold Shoulder Game

Cold Shoulder is a simple top-down driving game created using Pygame-ce.
It was created to help demonstrate the difficulty of following the road rules while sharing the road with cyclists.

## How To Setup

To run the game, you must have Pygame-ce installed:

```
pip install pygame-ce
```

It can then be run with:

```
python Cold_Shoulder_Game\code\main.py
```

## How To Play

Use the WASD keys to move the blue car and QE to toggle indicators. The objective is to reach the delectable sweet treat while following the road rules as closely as possible. You are graded on how well you follow the road rules.
